inquirer.prompt() function is used to prompt the user to enter name and age, two question are presented to the user asking about name and age, the input provided by the user are stored in answer object. The name and age are extracted from the 'answer' object using destructing method. Then we use if statement to check is the condition true if the input value of name and age is provided the blocks code will be run, other than that it will be run the else condition.
